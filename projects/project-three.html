CourseMateAI

A full-stack RAG study assistant with 5 autonomous learning modes and near-zero hallucination via document-grounded responses.

The Problem

Students face information overload when studying from lecture slides, research papers, and textbooks. Rote memorization doesn't build understanding, and there's no AI tool specifically designed to support active recall and spaced repetition.

Existing AI assistants hallucinate frequently when asked about specific course material, making them unreliable for studying. Students need a tool that stays grounded in their actual course materials.

Technologies Used

Built with a focus on RAG accuracy, hallucination prevention, and multiple learning mode support.

LangChain + LangGraph

Orchestration framework for multi-step RAG pipelines. State machines manage the 5 learning modes: Quiz, Summary, Flashcards, Explain, Practice.

FAISS

Efficient similarity search for document retrieval. Hierarchical navigable small world graphs for fast nearest-neighbor queries at scale.

Mistral AI

Open-source model for document-grounded generation. Lower hallucination rates compared to larger models when properly prompted with retrieved context.

Challenges Solved

Hallucination Prevention

Problem:

LLMs often generate plausible-sounding but factually incorrect responses when answering questions about course material.

Solution:

Strict document-grounding with source citations. Responses must cite the exact document chunks used. If no relevant documents exist, the model explicitly states "I don't know."

5 Learning Modes

Problem:

Each learning mode (Quiz, Summary, Flashcards, Explain, Practice) requires different retrieval strategies and prompting approaches.

Solution:

LangGraph state machines with mode-specific nodes. Each mode has its own retrieval query template, prompt engineering, and response format requirements.

Document Processing Pipeline

Problem:

Lecture slides, PDFs, and research papers have different structures. Naive chunking loses important context boundaries.

Solution:

Structure-aware chunking with sliding window overlap. Markdown conversion for slides preserves formatting. Metadata includes section headers for contextual retrieval.

What I Learned

RAG Architecture Trade-offs

Chunk size, embedding model choice, and retrieval threshold tuning all significantly impact answer quality. A/B testing with real users is essential.

Hallucination Mitigation

Citation enforcement and explicit uncertainty declarations are more effective than prompt engineering alone for reducing hallucinations in factual Q&A.

Learning Mode UX Design

Different learning modes have different cognitive goals. Quiz needs distractors, Summary needs hierarchy, Flashcards need cloze deletion support.
